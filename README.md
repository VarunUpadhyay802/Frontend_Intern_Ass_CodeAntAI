## 🚀 Installation Guide

1. **Clone the repository:**
```bash
git clone https://github.com/your-username/codeant-assignment.git
```

2. **Install dependencies:**
```bash
cd Frontend_Intern_Assignment_CodeAntAI
npm install
```

3. **Run the development server:**
```bash
npm run dev
```

4. **Build for production:**
```bash
npm run build
```
